## 🗨️​ Instalação

### Instalação Manual (Local)
1. Faça o download dos arquivos do repositório.
2. Mova a pasta do tema para o diretório de extensões do seu VS Code:
   - **Windows:** `C:\Users\seu-usuário\.vscode\extensions\`
   - **macOS / Linux:** `~/.vscode/extensions`
3. Reinicie o VS Code e selecione o tema.

## 💜​ Contribuindo e Customizando

Adoraria ver como você deixa o tema com a sua cara! <br> Sinta-se à vontade para sugerir melhorias ou criar sua própria variação.

### Como fazer o seu:
1. Faça um **Fork** deste repositório.
2. Clone o seu fork localmente: `git clone https://github.com/seu-usuario/seu-repositorio.git`
3. Crie uma branch para a sua feature ou modificação: `git checkout -b minha-customizacao`
4. Faça as alterações no arquivo de configuração do tema (geralmente `themes/Tema Querido-color-theme.json`).
5. Envie um **_Pull Request_** caso queira que suas melhorias sejam integradas ao tema principal!
