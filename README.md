# Tema Querido

Meu tema personalizado para o VS Code.

![Tema Querido](https://raw.githubusercontent.com/Pichau/tema-querido/main/assets/screenshot.png)

## Instalação

1. Abra o VS Code.
2. Clique em **Extensions** no canto superior esquerdo.
3. Procure por **Tema Querido** e clique em **Instalar**.
4. Clique em **Atualizar** para instalar o tema.

## Contribuindo

Você pode contribuir com o tema através de pull requests. Para fazer isso, siga os seguintes passos:

1. Crie um fork do repositório.
2. Clone o fork para o seu computador.
3. Crie uma nova branch para suas alterações.
4. Faça as alterações necessárias.
5. Commit suas alterações.
6. Crie um pull request para o repositório original.

Lembre-se de seguir as convenções de codificação e organização do repositório.
